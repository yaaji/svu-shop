Course Objectives

This course aims to teach the student how to develop web pages and scripting on the client side. It focuses on basic concepts such as HTML and CSS that are essential for designing and developing any web page. The course covers basic web page creation, cascading style sheets, and introduction to JavaScript.

Learning Outcomes

By the end of this course, the learner is expected to acquire and learn the following subjects:

  • XHTML Basics
  • advanced skills in XHTML
  • Cascade Style Sheets
  • JavaScript basics
  • HTML and JavaScript
  • Dynamic pages
  • JQuery basics
  • Attributes and methods in JQuery
  • JQuery skills
  • XML
